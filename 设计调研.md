## 最终确定的风格方向

### 整体风格: 3D + 2D 混合 | 温暖暗色 | 低多边形工作台

**Hero 区域**: 低多边形风格的 3D 工作台/房间场景 (React Three Fiber)
- 显示器（代表前端开发）
- 游戏手柄/UE 元素（代表游戏开发）
- 摄像机/拍板（代表视频创作）
- 画板/调色板（代表设计）
- 咖啡杯、书本等生活细节
- 鼠标可拖拽旋转，悬停物品高亮
- 柔和灯光 + 轻微动画

**其余页面**: 2D + 滚动动画，与 3D 场景风格统一的温暖暗色主题

## 技术栈

- **框架**: Astro 5.x (SSG)
- **3D**: React Three Fiber + @react-three/drei
- **交互组件**: React 19
- **样式**: Tailwind CSS v4
- **动画**: Framer Motion + GSAP
- **内容**: Astro Content Collections + MDX
- **部署**: Docker + Caddy (腾讯云)
